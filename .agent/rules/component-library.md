# UI |

When creating UI, first check /app/components/COMPONENTS.md to see if we may already have a component that you can use.

Secondly check if a relevant component exists in the Nuxt UI framework. We are using nuxt ui v4 so make sure you follow the documentation for that version. Docs can be found here:
https://ui.nuxt.com/llms.txt

And then create the component if needed.

When you create a component, be sure to document it in /app/components/COMPONENTS.md

Read /app/components/AGENTS.md for guidance when creating or updating components.

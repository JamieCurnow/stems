# Composables |

Take advantage of the existing composables in the app. Documentation can be found in /app/composables/COMPOSABLES.md.

First check /composables/COMPOSABLES.md for existing functionality.

Secondly check vueuse to see if the functionality exists there - @vueuse/core is installed.

Then finally create a new file in /composables and be sure to document it in /app/composables/COMPOSABLES.md.

Read /app/composables/AGENTS.md for guidance when creating or updating composables.

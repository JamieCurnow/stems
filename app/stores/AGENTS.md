# Pinia Stores

The files in this directory are pinia stores for the application used to store global state.

There is a STORES.md file that documents the stores that are available in the application.

- If you add a new store, it must be documented in STORES.md.
- If you remove a store, it must be removed from STORES.md.
- If you update a store, it must be updated in STORES.md.
- If you learn something worth preserving, add it to the `## Learnings` section in STORES.md.
